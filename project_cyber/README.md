# RiskLens AI Dashboard - Monorepo

This project is organized as a monorepo with separate frontend and backend directories.

## Project Structure

```
project_cyber/
├── frontend/           # Next.js frontend application
│   ├── app/
│   ├── components/
│   ├── package.json
│   ├── tsconfig.json
│   └── ...
├── backend/            # Backend API (Node.js/Express)
│   └── (To be initialized)
└── node_modules/       # (Shared dependencies)
```

## Getting Started

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

## Development

- **Frontend Developer**: Works in the `frontend/` directory (Next.js with React and TypeScript)
- **Backend Developer**: Works in the `backend/` directory

Each directory has its own `package.json` and dependencies.
